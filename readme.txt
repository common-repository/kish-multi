A plugin to manage multiple Wordpress blogs from one blog

== Description ==
This plugin is for people who maintain multiple blogs. You can use this plugin to manage all your blogs from a single blog. I have used ajax, so there is only one page to load and its fast and easy to use.

1. View, pending comments, posts
1. Moderate comments, reply to comments, delete comments
1. Read Post
1. Simple installation
1. This plugin needs to be installed only on one blog



== Installation ==

Installation is very simple, just upload the unzipped files to the plugins directory and activate the plugin. Make sure that you chmod the upload uploadify/uploads file to 777
